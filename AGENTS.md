## Design Context

See PRODUCT.md and DESIGN.md at the project root for the full brief. Key anchors:

- **Register:** brand (personal portfolio — design IS the product)
- **Core narrative:** AI-assisted software engineering mastery. The key differentiator: a **software engineer behind the wheel**, not a prompter. Design, development, and AI are interlocking skills, not separate hats.
- **Personality:** Confident + sharp (60%) / warm + approachable (40%)
- **Palette:** Dark-first, violet primary (`oklch(0.62 0.18 294)`), warm amber accent (`oklch(0.70 0.14 70)`), near-black bg
- **Fonts:** Sora (display, geometric sans) · Figtree (body) · Fira Code (mono)
- **Anti-references:** No gradient text, glassmorphism, numbered section markers, tiny uppercase eyebrows (only "Selected work" survives on Projects section), side-stripe borders, AI-template look, "AI wrapper" aesthetics
- **Accessibility:** WCAG AA, reduced-motion, visible focus, semantic HTML
- **Live mode configured:** `.impeccable/live/config.json` ready — run `/impeccable live` for in-browser variant iteration
